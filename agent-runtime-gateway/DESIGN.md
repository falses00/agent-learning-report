# Design

## Intent

这是可长期使用的工程学习控制台，不是营销落地页，也不是单篇长报告。视觉继承旧版 Agent Runtime Gateway 的蓝橙品牌语言、强标题和宽松阅读节奏；信息架构保留新版的进度、证据门禁、自测、演练和资料库。

桌面端用于边学边运行命令，移动端用于查看路线、复习阶段和完成自测。任何视觉装饰都不能压过“我在哪、下一步做什么、需要留下什么证据”。

## Information Architecture

```text
Top bar: 总览 / 阶段学习 / 故障演练 / 资料库
Left rail: 品牌、搜索、整体进度、分组资料、导出/重置
Overview: 品牌标题 -> 四段路线 -> 当前阶段 -> 起点与今日任务 -> 完整阶段地图
Stage: 五步学习闭环 -> 阶段轨道 -> 交付/事故/修复/证据 -> 自测与门禁
Mobile: 顶部目录 + 底部四视图导航
```

首屏必须同时回答：课程目标、当前阶段、四段主线和当前可运行边界。完整路线按四组呈现：

1. 工程起步：F0。
2. 核心运行时：S0-S2。
3. 可信生产系统：S3-S8。
4. 协作与发布：S9-S10。

## Visual Language

```css
:root {
  --bg: #f5f7fb;
  --surface: #ffffff;
  --surface-muted: #f2f5fa;
  --ink: #1a1d26;
  --ink-muted: #667085;
  --accent: #0052d9;
  --accent-hover: #003eb3;
  --accent-mid: #1a6fe8;
  --warning: #c96442;
  --success: #16845b;
  --danger: #c4413b;
}
```

- 蓝色表示主线、当前选择与可运行能力。
- 橙色表示阶段边界、提醒和后续实验。
- 渐变只用于品牌标记、当前阶段和短强调线，不作为大面积正文背景。
- 状态必须同时有文字，不能只依赖颜色。

## Typography

- 标题使用高字重系统无衬线字体，恢复旧版明确的品牌层级。
- 总览品牌标题桌面 44px、移动端 32px，只在总览首屏使用。
- 页面标题 28-32px，面板标题 17-24px，正文 14-16px。
- 长文行高 1.65-1.8，正文最大 82ch。
- 代码和阶段编号使用系统等宽字体。
- 字间距为 0，不按视口连续缩放字号。

## Teaching Components

- `JourneyStrip`：四段学习路线，显示阶段范围、完成数和当前所在段。
- `RoadmapGroup`：按四段组织 F0、S0-S10，避免一张长平表。
- `RouteCard`：阶段名称、能力关键词、状态和可运行边界。
- `ContinuePanel`：唯一当前阶段、下一动作和教学基线入口。
- `StageMethod`：理解边界、完成实现、正常测试、失败测试、保存证据。
- `StageRail`：12 个阶段快速切换，当前阶段使用品牌渐变。
- `EvidenceChecklist`：五项证据全部齐全后才允许本地标记通过。
- `FailureSimulator`：策略、审批、租户隔离和注入的前端教学演练。
- `DocumentReader`：单篇资料、来源、前后翻阅和站内链接路由。

## Layout

- 桌面左栏 308px，顶部栏 64px，主工作区最大 1120px。
- 1040px 以下四段路线改为两列，阶段卡片改为两列。
- 900px 以下左栏变为 drawer。
- 720px 以下四段路线横向浏览，完整阶段地图改为纵向时间线。
- 移动端五步闭环保持五列同时可见；12 阶段轨道允许横向滑动。
- 固定顶部栏和底部导航必须为正文预留空间。

## Radius And Depth

- 控件 radius 6px，面板和阶段卡片 radius 8px。
- 阴影仅用于区分可点击阶段、当前任务和阅读面板。
- 页面区段使用留白和分隔线，不把每个区段都做成浮动卡片。
- 禁止卡片嵌套卡片。

## Interaction

- 点击四段路线可跳到对应完整路线分组。
- 点击阶段卡进入阶段工作台，并保持当前阶段和进度。
- hover、focus-visible、active、disabled 状态必须清晰。
- 动效控制在 150-220ms；`prefers-reduced-motion` 下取消平滑滚动与模拟延迟。

## Accessibility And Safety

- 正文对比度至少 4.5:1，键盘焦点不能只靠颜色。
- 所有横向轨道必须保留可滚动区域且不能造成页面横向溢出。
- Markdown 渲染禁止脚本、iframe、事件属性和 `javascript:` URL。
- 外部 CDN 失败时退回纯文本 Markdown。
- `本地进度`、`教学模拟`、`不执行真实操作`边界始终可见。
