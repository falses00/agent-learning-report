## 4. 采用原则

本项目不会因为某个项目热门就直接引入依赖。

采用方式只有三种：

- `dependency allowed`：许可证清楚、边界清楚、可替换。
- `adapter candidate after license review`：有价值，但先审许可证、部署方式和商业条款。
- `pattern only`：只学习设计模式，不直接引入。
