## 7. 仍需二次核验

- 托管型项目的 self-host 能力、遥测默认值、许可证和商业条款。
- mcp-agent、mcp-use、AG2 等快速变化项目的 API 稳定性。

- 评测工具的成本控制，避免红队和 LLM-as-judge 失控。
