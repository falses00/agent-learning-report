## 6. 进入课程的规则

外部资料只有满足下面条件，才进入课程正文：

1. 能解释一个工业级失控风险。
2. 能映射到某个 Phase。
3. 有官方文档或代码仓库证据。
4. 能转成 Design-only 练习。
5. 能转成后续 Implementation-later 验收。

否则只放在证据库，不进入主线。
