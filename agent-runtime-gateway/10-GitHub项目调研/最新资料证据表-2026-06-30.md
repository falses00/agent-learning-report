## 4. 持续核验规则

进入代码实现或依赖引入前，必须重新核验：

- 官方文档。
- GitHub latest release。
- license。
- 安全公告。
- 是否 archived 或 maintenance mode。
- 是否需要云服务或外部账号。
- 是否会把企业数据传出本地环境。
