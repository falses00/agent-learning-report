## 3. 进入教学路线的处理方式

这些资料不直接变成“照着抄的方案”，只进入三个位置：

1. 课程失控风险：说明为什么某一层必须存在。
2. Design-only 练习：把外部经验转成设计卡、评审问题、失败样例。
3. Implementation-later 门禁：将来要写代码时，转成测试、评测、trace、audit 或发布门禁。
