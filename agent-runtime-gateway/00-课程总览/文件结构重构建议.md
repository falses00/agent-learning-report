## 5. 索引规则

以后凡是目录内文件超过 8 个，必须新增一个 `00-索引.md` 或 `README.md`，并写清：

- 先读哪 3-5 个。
- 哪些是课程。
- 哪些是模板。
- 哪些是证据。
- 哪些是后续工程参考。
- 哪些不适合新手直接读。

## 6. 停止条件

不要在没有用户确认前做以下操作：

- 批量移动文件。
- 批量改名。
- 删除历史反思文档。
- 创建源码目录。
- 把证据 JSON 改造成课程文档。

当前最稳妥路线是“保留索引和地图不动，先正式开 Design-only 主线课；进入代码阶段前再统一命名或移动文件”。
