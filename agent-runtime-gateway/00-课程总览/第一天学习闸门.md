## 4. 第一天不能进入下一步的情况

只要出现下面任一情况，就停在 Phase 0.1：

- 你只能复述“Runtime、Gateway、Policy”这些词，但说不出业务事故。
- 你不知道为什么模型只能提出工具申请，不能直接执行。
- 你不知道 trace、audit、eval 分别解决什么。
- 你说不清聊天 demo 和工业级 Agent 的差别。

## 5. 第一天完成句式

你能说出下面这段，就可以进入 Phase 1：

```text
我现在知道工业级 Agent 不是聊天窗口。
在工单退款场景里，模型可以建议，但不能直接执行退款。
执行要经过 Gateway、Policy、审批、Audit 和 Eval。
如果系统中断，只保存聊天记录不够，还需要 Runtime 状态和 checkpoint。
```
