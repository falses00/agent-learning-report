# v0.4 低保真可点击原型

状态：静态教学原型  
技术：纯 HTML / CSS / JavaScript  
边界：无构建工具、无依赖、无后端、无真实执行
渲染：已通过本机 Chrome 桌面与移动视口 QA

## 打开方式

直接打开：

```text
index.html
```

本目录不需要：

- `package.json`
- `node_modules`
- dev server
- 数据库
- API key
- 真实后端

目录内 `图像素材/` 是原型自包含参考图，`浏览器QA截图/` 是渲染验收证据。

## 覆盖范围

首轮 6 页：

1. C 首页 / 学习路线地图
2. A 全链路分层控制台
3. B 事故推演工作台
4. RAG 问题诊断器
5. LLM-Wiki 知识层
6. 证据与门禁页

覆盖 13 个 P0 验证主题：

- 建议与执行分离
- Runtime 状态
- Tool Gateway
- 长线任务
- RAG 找不到
- RAG 找错租户
- RAG 引用可信
- 记忆污染
- Eval 门禁
- Trace/Audit
- 沙箱风险
- 多 Agent 协同
- LLM-Wiki 知识层

## 原型交互

- 左侧切换 6 个页面。
- 顶部切换默认、坏设计、低分回流状态。
- RAG 页面切换三类 RAG 问题。
- LLM-Wiki 页面切换知识治理步骤。
- 复述评分 0-5 会改变低分回流状态。
- 底部显示当前页面覆盖的 P0 主题。

## 不代表什么

这个原型不代表：

- 生产系统。
- 真实发布。
- 真实审批。
- 真实退款。
- 真实工具调用。
- 真实凭据。
- 真实企业数据。

它只用于学习：

```text
事故 -> 负责层 -> 最小证据 -> 复述 -> 低分回流
```

## 验收方式

打开任一页面后，学习者必须能回答：

1. 当前 Phase 是什么？
2. 这页防什么事故？
3. 负责层是谁？
4. 最小验收证据是什么？
5. 复述低于 3 分时回流到哪里？
6. 为什么这不是生产控制台？
